About Hospital Management System

A comprehensive web application for managing hospital operations efficiently.

Features

  • Admin Portal: Manage doctors, patients, and appointments
  • Doctor Portal: View appointments, add treatments, manage availability
  • Patient Portal: Book appointments, view medical history
  • REST API: JSON endpoints for integration
  • Email Notifications: Automatic appointment confirmations

Technology Stack

  • Backend: Flask, SQLite
  • Frontend: HTML, CSS, Bootstrap 5
  • Authentication: Flask-Login
  • Visualization: Chart.js
